# Uma URI (Uniform Resource Identifier) é uma sequência de caracteres que identifica um recurso físico ou lógico.
# Existem dois tipos de URI: URL e URN.
# Uma URL (Uniform Resource Locator) é a sequência de caracteres colocada depois do URI para localizar o recurso.
# Uma URN (Uniform Resource Name) é a sequência de caracteres colocada depois do URI para localizar o recurso.

# Toda URL também é uma URI, mas não vice versa.
